Detailed Description

SpeedSkating class is the superclass of the 3 SpeedSkating events.

DEFINTION: Skating done on ice at a fast pace around an oval track utilizing inline skates where scoring is based upon time. One of the two main sporting categories that consist of three unique events which athletes participate separately.
