Detailed Description

SS500m is one of the 3 SpeedSkating events.

DEFINITION: Form of speed skating where athletes utilize inline skates and race around a 400m rink 1.25 times. Whichever athlete crosses the finish line first has the lowest time.

CONSTRAINTS: A maximum of four athletes compete at a given time.
